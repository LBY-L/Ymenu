# Ymenu

A compilation of many scripts in a single application for ArchLinux

# Install guide

```
cd Ymenu
```
```
chmod +777 Install
```
```
source Install -Modes
```

Modes:

-e  To install ymenu in English

-s  To install ymenu in Spanish

# Usege

`ymenu --help` to list the options
